# React with Webpack Setup

## Install modules

```
npm install --save react react-dom
npm install --save-dev webpack webpack-cli webpack-dev-server
npm install --save-dev babel-loader @babel/core @babel/preset-env @babel/preset-react
```

## DEMO

```
npm run start
```
